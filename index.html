What I do

Landing Page

A landing page is the page a visitor arrives at on your website after clicking an ad .A landing page should offer all the necessary information, but not so much as to overwhelm the visitor.

Responsive-Website

Responsive web design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes. Recent work also considers the viewer proximity as part of the viewing context as an extension for RWD. Content, design and performance are necessary across all devices to ensure usability and satisfaction.

Backend-Development

Backend development handle the ‘behind-the-scenes’ functionality of web applications. It’s code that connects the web to a database, manages user connections, and powers the web application itself. Backend development works in tandem with the front end to deliver the final product to the end user.
